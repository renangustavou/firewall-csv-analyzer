1. Visão Geral

O Firewall CSV Analyzer é uma ferramenta baseada em web para analisar arquivos CSV contendo informações de tráfego de rede. Ele categoriza os registros em Permissões (Allowlist), Sinalizações (Watchlist) e Bloqueios (Blocklist) com base em regras definidas, exibindo os resultados em um layout de dashboard interativo. A aplicação também permite que os usuários acompanhem o progresso da análise em tempo real através de uma barra de progresso.

2. Pré-requisitos

Para rodar este projeto localmente, você precisará das seguintes ferramentas:

  • Node.js (opcional, para usar live-server ou outro servidor local)
  • Python (opcional, para usar SimpleHTTPServer)

Navegadores compatíveis:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

5. Descrição das Funcionalidades

  • Seleção de Arquivo CSV: O usuário pode selecionar um arquivo CSV de tráfego de rede para análise.
  • Análise de Dados: O arquivo CSV é processado para identificar registros que correspondem a padrões suspeitos definidos em rules.js.
  • Exibição dos Resultados: Os registros são categorizados como Permissões (Allowlist) ou Bloqueios (Blocklist) e exibidos em tabelas separadas.
  • Barra de Progresso: A barra de progresso mostra em tempo real o percentual de conclusão da análise do arquivo.
  • Botões de Limpeza: Permitem que o usuário limpe as listas de permissões e bloqueios.

6. Instruções de Uso

  1. Abrir o aplicativo em um navegador.
  2. Clique no botão "Selecione o arquivo CSV" para escolher o arquivo de tráfego que deseja analisar.
  3. Acompanhe o progresso da análise através da barra de progresso.
  4. Visualize os resultados nas tabelas de Permissões e Bloqueios.
  5. Limpe as listas usando os botões "Limpar Permissões" e "Limpar Bloqueios" conforme necessário.

7. Customização

Regras de Análise:

As regras de análise podem ser customizadas no arquivo rules.js. Edite as variáveis e funções para ajustar as regras de acordo com suas necessidades.

const suspiciousPorts = [21, 22, 80, 443, 3306];
const suspiciousMethods = new Set(['POST', 'PUT', 'DELETE', 'CONNECT', 'PATCH']);;
const suspiciousCountries = new Set(['CN', 'RU', 'KR', 'IR', 'IN']);
const suspiciousDevices = new Set(['mobile', 'unknown']);
const suspiciousUserAgents = new Set(['bot', 'spider', 'crawler', 'scanner']);
const suspiciousReferers = new Set(['http://www.hernandez.com/', 'http://untrustedsite.com/']);
const suspiciousURIKeywords = new Set(['login', 'admin', 'wp-login', 'exploit', 'shell']);
